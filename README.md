## 项目概述

文字格斗游戏是一款完全运行在控制台（终端）的Java程序。它包含了完整的用户管理（登录注册）和核心战斗循环。玩家可以通过分配属性点来定制自己的英雄，并在与五种不同特点的敌人的战斗中，运用策略和技能，体验角色成长的乐趣。

**主要特点**：
- ✅ **完整的账户系统**：注册、登录、密码错误锁定、忘记密码重置
- ✅ **自定义角色创建**：自由分配100点属性点
- ✅ **五种独特敌人**：每种敌人拥有不同的属性与特殊技能
- ✅ **四种玩家技能**：包括普通攻击、强力一击、生命/魔力汲取
- ✅ **动态成长系统**：玩家每3胜获得属性提升，敌人每场战斗后也会变强
- ✅ **道具系统**：6种道具可在战斗中使用，战后有几率获得

## 技术栈

- **语言**: Java (版本 8 或更高)
- **核心API**: Java集合框架 (`ArrayList`, `Random` 等), `Scanner` 控制台交互
- **开发环境**: IntelliJ IDEA (或其他Java IDE)
- **版本控制**: Git

## 功能模块

### 用户系统
游戏启动后首先进入登录注册界面（`Login.java`）。

- **用户数据模型 (`User.java`)**:
  - `id`: 自动生成，格式为 `nangua` + 5位随机数字
  - `username`: 唯一，3-16位，字母数字组合，不能全数字
  - `password`: 3-8位，必须包含字母和数字
  - `phone`: 11位，以1开头的纯数字
  - `status`: `true` 为可用，连续输错3次密码后变为 `false`（锁定）

- **核心功能**:
  - **注册**: 对用户名、密码、手机号进行格式校验，密码需二次确认
  - **登录**: 验证用户名和密码。使用验证码（4字母+1数字）防止自动化爆破。连续输错3次将锁定账号
  - **忘记密码**: 通过用户名和预留手机号验证身份后，可重置密码

### 角色创建
登录成功后，进入角色创建界面（`Fighting.java` - `createPlayer` 方法）。

- **属性分配**: 玩家共有100点属性点，可自由分配到以下四个属性：
  - **生命值**: 基础 50 + (分配点数 × 10)
  - **魔力值**: 基础 30 + (分配点数 × 5)
  - **攻击力**: 基础 10 + (分配点数 × 3)
  - **防御力**: 基础 0 + (分配点数 × 1)

- **初始技能**: 创建完成后，角色自动获得四个基础技能：
  - 普通攻击、强力一击、生命汲取、魔力汲取

### 战斗系统
游戏的核心是回合制战斗，代码主要在 `Fighting.java` 中。

#### 玩家技能

| 技能名称 | 消耗 | 效果与说明 |
| :--- | :--- | :--- |
| 普通攻击 | 无 | 造成 `攻击力 - 防御力` 的基础伤害 |
| 强力一击 | 30 MP | 造成 `(攻击力 - 防御力) × 2` 的高额伤害 |
| 生命汲取 | 20 MP | 恢复自身最大生命值 **10% - 50%** 的随机生命值 |
| 魔力汲取 | 50 HP | 恢复自身最大魔力值 **25% - 50%** 的随机魔力值 |

#### 敌人类型

| 敌人名称 | 生命值 | 攻击力 | 防御力 | 特殊技能 | 技能效果 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 初级战士 | 160 | 25 | 20 | 猛击 | 造成200%的普通攻击伤害 |
| 敏捷刺客 | 100 | 40 | 10 | 快速攻击 | 连续进行两次攻击，每次造成80%伤害 |
| 重装坦克 | 200 | 20 | 30 | 防御姿态 | 进入防御状态，接下来 **2回合** 内受到的伤害减半 |
| 神秘法师 | 120 | 35 | 15 | 斗转星移 | 激活吸血状态，接下来 **2次** 攻击回复本次伤害30%的生命值 |
| 游猎射手 | 130 | 30 | 15 | 蓄势待发 | 本回合造成90%伤害，但下一回合普通攻击伤害翻倍 |

#### 道具系统

| 道具名称 | 效果 | 使用时机 |
| :--- | :--- | :--- |
| 桃子 | 恢复100点生命值 | 战斗中使用 |
| 煎蛋 | 最大生命值永久+20% | 战斗中使用 |
| 花酿鸡 | 当前回合受到的伤害减半 | 战斗中使用 |
| 黄酒 | 下回合造成的伤害翻倍 | 战斗中使用 |
| 黑背鲈鱼 | 恢复100点魔力值 | 战斗中使用 |
| 白玉汤 | 最大魔力值永久+20% | 战斗中使用 |

- **战斗规则**:
  1. 每场战斗开始前，从敌人列表中随机选取一名对手
  2. **每回合**，会显示敌我双方的血条（`█`）和蓝条
  3. **玩家回合**：选择技能（1-4）或使用道具（5）进行攻击或恢复
  4. **敌人回合**：敌人有70%概率普通攻击，30%概率使用其专属技能
  5. 回合持续进行，直到一方生命值归零。玩家获胜则进入结算，失败则游戏结束

- **状态机制**:
  - **防御姿态**: `defending=true` + 计数器，每回合结束减1，归零时状态关闭
  - **斗转星移**: `bloodsucking=true` + 计数器，每次攻击减1，归零时状态关闭，回复伤害的30%
  - **蓄势待发**: `attacking=true`标记，下回合普通攻击翻倍后自动关闭

### 成长系统
为了让游戏更有深度，加入了动态成长机制。

- **玩家成长**:
  - 每取得 **3场** 胜利（且不为新手教学关），玩家基础属性会获得一次提升：
    - 最大生命值 +100
    - 最大魔力值 +50
    - 攻击力 +20
    - 防御力 +10

- **战后恢复**:
  - **生命值**：恢复已损失生命值的 **50% - 80%**
  - **魔力值**：恢复最大魔力值的 **40% - 60%**

- **敌人成长**:
  - 每次玩家获胜后（新手教学关除外），所有敌人的属性都会增强：
    - 最大生命值 +100
    - 攻击力 +10
    - 防御力 +5
    - 生命值回满，重置防御状态

- **通关条件**:
  - 累计获得 **10场** 胜利即为通关，游戏将显示祝贺信息

- **新手教学**:
  - 第一场胜利（`win == 0`）时显示新手教学通过提示

## 核心类结构

项目采用面向对象设计，核心类关系如下：

```
├── APP.java                  # 程序入口
├── domain                    # 实体类包
│   ├── Character.java        # 角色基类（生命、攻击、防御、状态等）
│   ├── Hero.java             # 英雄类（继承Character，增加魔法值、技能、背包）
│   ├── Enemy.java            # 敌人类（继承Character，增加技能、吸血状态）
│   ├── User.java             # 用户类（登录注册信息）
│   └── Consumable.java       # 道具类（名称、数量）
└── ui                        # 界面与逻辑包
    ├── Login.java            # 登录注册界面及验证逻辑
    └── Fighting.java         # 核心战斗逻辑、角色创建、成长系统
```

- **`Character`** 作为基类，封装了所有角色的通用属性和行为（如受伤害、恢复、状态管理）
- **`Hero`** 和 **`Enemy`** 继承自 `Character`，并分别扩展了各自特有的属性和方法
- **`Login`** 类负责所有与用户账户相关的控制台交互和验证
- **`Fighting`** 类包含了主要的游戏循环、战斗逻辑、数值计算和成长机制
